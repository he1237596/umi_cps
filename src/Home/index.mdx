## 蘑菇物联前端体系之--前端组件库
### 安装
``` js
npm install umi_cps
// 或者用yarn安装
yarn add umi_cps
```

### 使用
``` jsx
import { 
  MgButton,
} from 'umi_cps'
const Demo = () => <MgButton>蘑菇的第一个button</MgButton>
export default Demo
```
### 实例

    <MgButton bordered={true} type='ghost'>蘑菇的第二个按钮</MgButton>
